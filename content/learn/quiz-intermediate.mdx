# Intermediate *checkpoint.*

Five questions. These go deeper — each one requires you to apply a concept from chapters 7–12, not just recall a definition.

---

<Section n={1} title="Count the faan">

East round. You're seated South. You win by self-draw:

<TileRow tiles={[
  { suit: 'dot', value: 2 },
  { suit: 'dot', value: 3 },
  { suit: 'dot', value: 4 },
  { suit: 'dot', value: 5 },
  { suit: 'dot', value: 6 },
  { suit: 'dot', value: 7 },
  { suit: 'bamboo', value: 1 },
  { suit: 'bamboo', value: 2 },
  { suit: 'bamboo', value: 3 },
  { suit: 'wind', value: 'east' },
  { suit: 'wind', value: 'east' },
  { suit: 'wind', value: 'east' },
  { suit: 'character', value: 5 },
  { suit: 'character', value: 5 },
]} gap="tight" />

<Callout title="Answer — 3 faan">
Common hand (平糊, all chows + non-honor pair) = **1 faan**. East wind pung in East round = **+1 faan** (round wind bonus). You're South seat, so East is not your seat wind — only one bonus applies. Self-draw = **+1 faan**. Total: **3 faan**.
</Callout>

</Section>

<Section n={2} title="Best discard?">

You hold these 13 tiles. Which should you discard?

<TileRow tiles={[
  { suit: 'dot', value: 3 },
  { suit: 'dot', value: 4 },
  { suit: 'bamboo', value: 6 },
  { suit: 'bamboo', value: 7 },
  { suit: 'bamboo', value: 8 },
  { suit: 'character', value: 1 },
  { suit: 'character', value: 2 },
  { suit: 'character', value: 3 },
  { suit: 'character', value: 7 },
  { suit: 'character', value: 8 },
  { suit: 'character', value: 9 },
  { suit: 'wind', value: 'north' },
  { suit: 'dot', value: 9 },
]} gap="tight" />

<Callout title="Answer — North Wind or 9 Dot">
Three complete groups (6–7–8 Bamboo, 1–2–3 Characters, 7–8–9 Characters) and one two-sided chow fragment (3–4 Dots, waiting on 2 or 5). North Wind and 9 Dot are isolated — neither connects to anything. Discard either. The 3–4 Dots fragment is worth 8 outs; never discard it over an isolated tile.
</Callout>

</Section>

Tenpai? If yes, what are your outs?

<Callout title="Answer — Yes. Pair wait on 7 Dot.">
Four complete melds: 1–2–3 Dots (chow), 5 Bamboo × 3 (pung), 4–5–6 Characters (chow), Green Dragon × 3 (pung). The 7 Dot is the only remaining tile — it needs a pair. You're in tenpai with a **pair wait (單釣)**: at most 3 remaining 7 Dots in the wall. Weak wait, but tenpai.
</Callout>

<Section n={4} title="Should you call this pung?">

An opponent discards South Wind. You hold two South Winds. It's turn 5. You're seated North. The current round is East. Call?

<Callout title="Answer — Probably not">
South Wind is neither your seat wind (North) nor the round wind (East) — the pung adds 0 bonus faan. Calling exposes a set, telegraphs your hand, and costs flexibility for zero faan gain. With ~17 wall tiles yet to draw, you have a good chance to draw the third South naturally. Only call if you're committed to an all-pungs hand and already have other exposures.
</Callout>

</Section>

<Section n={5} title="What does this river tell you?">

Player to your left discarded in order: East Wind, West Wind, North Wind, 3 Characters, 9 Characters. They have one exposed set: a pung of Green Dragon.

What hand type are they likely building?

<Callout title="Answer — Mixed one-suit, probably Dots or Bamboo">
Three winds and two Characters discarded early — they've cleared both honor singletons and one suit. With a Green Dragon pung exposed, they're keeping dragons plus one suit. Pattern: **mixed one-suit**. Mid-tiles in Dots or Bamboo are most likely to feed their hand. Characters they've already thrown are safe to throw back.
</Callout>

</Section>
